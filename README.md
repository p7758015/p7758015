# AI Engineer / LLM Developer

Разрабатываю AI-ассистентов и нейроагентов на Python под бизнес-задачи любой сложности: от автоматизации контента и клиентской поддержки до анализа коммуникаций и других процессов, которые можно масштабировать с помощью AI.

Работаю с LLM-системами: мультишаговая логика, tool calling, RAG, интеграции с API и пользовательские интерфейсы (Telegram-боты, веб-сервисы). Также использую AI-first подход к разработке (vibe coding, Cursor) для ускорения создания продуктов.

## 🚀 Основные проекты

### 1️⃣ AI Call Auditor (Анализ звонков менеджеров)

**Problem**
Руководителям сложно быстро и объективно оценивать качество звонков менеджеров.

**Solution**
AI-сервис анализирует разговор и:

* оценивает качество звонка (0-100%)
* выявляет сильные и слабые стороны
* формирует структурированный отчёт
* легко интегрируется с CRM и системами телефонии

Python, LLM, Docker, FastAPI

### 4️⃣ Job Gap Assistant (AI-анализ навыков и вакансий)

**Problem**
Кандидатам сложно понять, насколько они подходят под вакансию и что нужно подтянуть.

**Solution**
AI-сервис:

* анализирует описание вакансии
* оценивает соответствие кандидата (fit score)
* выявляет пробелы в навыках
* предлагает проекты для прокачки
* формирует roadmap развития на 2–4 недели

Проект реализован с использованием **vibe coding (Cursor AI)** - демонстрирует современный подход к быстрой разработке AI-сервисов.

**Stack**
Python, FastAPI, LLM, Pydantic, Cursor (AI IDE)
